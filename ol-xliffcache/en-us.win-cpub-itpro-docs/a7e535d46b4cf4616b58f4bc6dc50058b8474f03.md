# DELETE APP


Removes an application record from the file system cache to make it no longer visible. Usersâ€™ shortcuts and file type associations are hidden but not deleted. No user settings are removed.

`SFTMIME DELETE APP:application [/LOG log-pathname | /CONSOLE | /GUI]`
