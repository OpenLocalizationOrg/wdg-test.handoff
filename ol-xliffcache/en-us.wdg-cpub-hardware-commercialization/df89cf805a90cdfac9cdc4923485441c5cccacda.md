# Stacks


Represents a collection of stacks. If the **Operation** attribute is specified, the **Stack** elements can be set or added to the collection.

## Syntax


``` syntax
<Stacks Operation = OperationEnumeration>

  <!-- Child elements -->
  Stack

</Stacks>
```
