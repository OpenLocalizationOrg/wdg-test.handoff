# Clone


Creates a clone enumerator.

## Syntax


``` syntax
HRESULT Clone
  ([out] IEnumProfile** ppEnum)
;
```

## Parameters


<a href="" id="ppenum"></a>*ppEnum*  
\[out\] Returns a pointer to the location of the clone enumerator.
