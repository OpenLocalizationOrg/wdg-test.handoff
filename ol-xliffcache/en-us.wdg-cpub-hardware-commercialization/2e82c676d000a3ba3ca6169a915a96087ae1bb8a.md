# GetText


Obtains the specified text.

## Syntax


``` syntax
HRESULT GetText
  ([in] ULONG iText,
  [out] BSTR* pbstrText)
;
```

## Parameters


<a href="" id="itext"></a>*iText*  
\[in\]

<a href="" id="pbstrtext"></a>*pbstrText*  
\[out\] A pointer to the text string.
