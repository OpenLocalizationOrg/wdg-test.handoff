# EventCollectorId


Represents an event collector identifier.

## Element Hierarchy


&lt;[WindowsPerformanceRecorder](windowsperformancerecorder.md)&gt;

``` syntax
<EventCollectorId Value = IdType>

  Buffers,
  StackCaching,
```
