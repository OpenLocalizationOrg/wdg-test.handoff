# Clone


Clones the whole set of errors and warnings.

## Syntax


``` syntax
HRESULT Clone
  ([out] IEnumControlWarningInfo** ppEnum)
;
```

## Parameters


<a href="" id="ppenum"></a>*ppEnum*  
\[out\] On return, a pointer to the location of the clone enumerator.
