# winlogon


This action produces a text file that summarizes metrics related to logon.

``` syntax
-a winlogon [-range T1 T2]
```

## Options


<a href="" id="-ranget1-t2"></a>**-range***T1 T2*  
Shows events from *T1* to *T2*, in microseconds.
