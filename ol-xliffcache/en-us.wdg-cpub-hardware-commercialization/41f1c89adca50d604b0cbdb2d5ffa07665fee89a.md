# WaitForText


Waits until the user adds the appropriate text strings and calls SetTextAvailable.

## Syntax


``` syntax
HRESULT WaitForText
  ([in] ULONG Milliseconds)
;
```

## Parameters


<a href="" id="milliseconds"></a>*Milliseconds*  
\[in\] Indicates the number of milliseconds for the wait delay.
