# ProblemCategories


Represents a collection of problem categories. This element is for internal use only.

## Element Hierarchy

     &lt;[Profiles](profiles.md)&gt;

          &lt;[Profile](profile-wpr.md)&gt;

## Syntax


``` syntax
<ProblemCategories Operation = "Set" | "Add" | “Remove” >

  <!-- Child elements -->
  ProblemCategory

</ProblemCategories>
```
