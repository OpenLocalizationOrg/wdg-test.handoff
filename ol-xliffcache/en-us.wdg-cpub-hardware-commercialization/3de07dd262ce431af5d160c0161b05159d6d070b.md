# Remove


Removes an [IProfile](iprofile.md) object from the collection.

## Syntax


``` syntax
HRESULT Remove
  ([in] IProfile* pProfile)
;
```

## Parameters


<a href="" id="pprofile"></a>*pProfile*  
\[in\] A pointer to the **IProfile** object to be removed from the collection.
