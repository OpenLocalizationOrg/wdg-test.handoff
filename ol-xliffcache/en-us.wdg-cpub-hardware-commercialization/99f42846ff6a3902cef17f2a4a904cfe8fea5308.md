# GetFileName


Obtains the string from the specified file.

## Syntax


``` syntax
HRESULT GetFileName
  ([out] BSTR* pbstrFileName)
;
```

## Parameters


<a href="" id="pbstrfilename"></a>*pbstrFileName*  
\[out\] A pointer to the string file name.
