# GetHResult


Returns an HRESULT value that indicates the error code.

## Syntax


``` syntax
HRESULT GetHResult
  ([out, retval] HRESULT* pHResult)
;
```

## Parameters


<a href="" id="phresult"></a>*pHResult*  
\[out\] Pointer to the HRESULT value that indicates the error code.
